Login

From here you can login to gain full access with your credentials.

Logout

Logout by clearing caches

Exit

Orders Page

Client ID

Prior to creating a new order the client id must be set by selecting a company from the dropdown menu.

New Order

The Create New Order button will take you to the Catalogue Page where you can select items from our extensive catalogue.


Catalogue Page

Widget Navigation

The current widget image is show along with it's id, description and asking price. You can navigate over MegaMax's available widget via the previous and next arrow buttons.

User Fields

The Quantity fields must be set prior to Add To Order. The Discount price is optional.


Map Page

Display global map, initially focusing on the devices location. Then as each of the days orders are placed the map will be updated with appropriate markers.

There is additional touch gesture implemented whereby you can navigate around the map, zoom in and out on current location and reset map window to current location.
